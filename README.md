# Easy-Cleaner
一个由 AI 辅助开发的智能桌面归类与 C 盘安全瘦身工具
# 🚀 Smart System Helper (智能系统助手)

这是一个没有任何编程基础的创作者，在 Gemini AI 的辅助下完成的第一个开源工具。

### 🌟 核心功能
- **一键桌面归类**：自动识别图片、文档、视频等后缀，让乱糟糟的桌面瞬间整洁。
- **安全撤销逻辑**：移动错了？一键还原，不给你的文件安全留隐患。
- **C 盘安全瘦身**：深度扫描 C 盘大文件，自动避开 Windows 系统敏感区，防止误删。
- **多线程扫描**：界面不卡死，流畅的交互体验。

### 🛠️ 开发背景
本项目展示了 AI 如何赋能普通人将创意变为现实。从最初的桌面整理脚本，到带 GUI 界面的成品软件，所有的逻辑迭代均由 AI 辅助完成。

### 📦 如何运行
1. 下载 `SmartOrganizer.py`。
2. 安装 Python 环境。
3. 运行脚本，或直接下载 Release 页面打包好的 `.exe` 文件。
版本号：v5.5

更新摘要：

    深度设计支持：新增对 PSD, PRPROJ, DWG, 3DS 等专业软件文件的智能分类。

    文件分流技术：同一大类（如文档）下，自动按 Excel/Word/PPT 创建子文件夹，拒绝大锅饭式收纳。

    安全扫描 2.0：不仅跳过 Windows，还增加了 ProgramData 和 AppData 的过滤，扫描速度相比旧版提升约 300%。

    容错处理：增加了重名文件自动改名（new_xxx）逻辑，防止整理时覆盖重要文件。
文档自动分流：Excel/Word/PPT 现在拥有独立的二级子文件夹。

影音分离：视频与音频不再共用文件夹，独立存储。

多级目录支持：引入了 mkdir(parents=True) 逻辑，收纳更优雅。
